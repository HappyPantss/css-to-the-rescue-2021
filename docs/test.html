Thema: Lay-out

CSS Shapes/ Vormen

De huidige implementatie van CSS Shapes is CSS Shapes Module Level 1, die voornamelijk draait om de shape-outside eigenschap. shape-outside definieert een vorm waar tekst omheen kan stromen.

Aangezien er een shape-outside eigenschap is, zou je kunnen aannemen dat er een overeenkomstige shape-outside eigenschap is die tekst binnen een vorm zou bevatten. Een shape-outside eigenschap kan in de toekomst realiteit worden, maar het is momenteel een concept in CSS Shapes Module Level 2 en wordt door geen enkele browser geïmplementeerd.

Basis vorm functies

We kunnen allerlei basisvormen in CSS definiëren door de volgende functiewaarden toe te passen op de shape-outside eigenschap:

  • circle()
  • ellipse()
  • inset()
  • polygon()

Transparante cirkel (Circle)

We beginnen met de circle() functie. Stel je een situatie voor waarin we een cirkelvormige avatar hebben van een auteur die we naar links willen laten zweven, en we willen dat de beschrijving van de auteur er omheen loopt. Gewoon een border-radius: 50% gebruiken van het avatar-element is niet genoeg om de tekst een ronde vorm te geven; de tekst behandelt de avatar nog steeds als een rechthoekig element.

Met de cirkelvorm kunnen we laten zien hoe tekst rond een cirkel kan vloeien.

Zoals je kunt zien, loopt de tekst rond de cirkelvorm, maar we zien eigenlijk geen enkele vorm. Als je de muisaanwijzer op het element in Developer Tools plaatst, zie je de werkelijke vorm die wordt ingesteld.
